### Oque eu aprendi

Nesse projeto aprendi a manipular as imagens de uma forma melhor, conseguindo recortar as imagens com o object position e a posicionar com position, como no exemplo abaixo:

```css
.main .best-time img {
    width: 25rem;
    height: 34rem;
    position: absolute;
    top: 12rem;
    left: 5rem;
    object-position: 2rem;
    object-fit: cover;
}
```
